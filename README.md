# photo_sketch_detection
Применение методов лицевой биометрии в криминалистике

Используются методы лицевой биометрии для поиска фотопортретов по скетчу на основе автоматической генерации «популяции скетчей». Для теста системы и оценки качества и точности результатов используются признаковые пространства «DFT» и «DCT», а также индекс структурного подобия «Structural SIMilarity Index» («SSIM»), для сравнения исходных фотопортретов, изображений из популяций и средним значением популяций.
